### 2. 为什么用向量/矩阵？

```python
# 传统方式：循环处理每个数据
prices = [150, 210, 270, 330]
doubled = []
for p in prices:
    doubled.append(p * 2)

# 向量方式：一次性处理所有数据
import numpy as np
prices = np.array([150, 210, 270, 330])
doubled = prices * 2  # 更快、更简洁！
```

```
   |              (拉伸 2 倍)          |
                                      

旋转 90 度：
⎡0  -1⎤
⎣1   0⎦
```

## 三、微积分：优化的引擎

### 1. 导数的直觉理解

```
导数 = 变化率 = 斜率

      y
      ↑
      │     ╱
      │   ╱
      │ ╱← 斜率 = 导数
      │╱
──────┴────────→ x

导数 > 0: 函数上升
导数 < 0: 函数下降
导数 = 0: 可能是极值点
```

### 2. 梯度：多变量的导数

```
梯度 = 指向函数增长最快的方向

        山顶
         ▲
         │
    梯度方向│
         │
你在这里 ●──────→ 往这个方向走最快到达山顶

梯度下降：反方向走，最快到达山谷（最小值）
```

### 3. 梯度下降算法

```
目标：找到损失函数的最小值

算法步骤：
1. 随机选择起点
2. 计算当前点的梯度
3. 沿梯度反方向走一小步
4. 重复 2-3，直到到达最低点

可视化：
损失
 ↑
 │  ╲
 │   ╲
 │    ╲
 │     ╲___
 │         ╲___
 │             ╲______
 └────────────────────→ 参数
     ● → ● → ● → ● (逐步接近最小值)
```

### 4. 学习率：步长很重要

```
学习率太大：
损失
 ↑    ●           ●
 │   ╱ ╲         ╱ ╲
 │  ╱   ╲       ╱   ╲  (来回震荡，不收敛)
 │ ╱     ╲     ╱     ╲
 └────────────────────→

学习率太小：
损失
 ↑
 │  ╲
 │   ╲●→●→●→...  (走得太慢)
 │    ╲
 └────────────────────→

学习率刚好：
损失
 ↑
 │  ╲
 │   ╲●→●→●→● (完美！)
 │    ╲
 └────────────────────→
```

### 5. 代码示例

```python
# 梯度下降找最小值
def gradient_descent():
    x = 10  # 初始位置
    learning_rate = 0.1
    iterations = 100
    
    for i in range(iterations):
        gradient = 2 * x  # f(x) = x² 的导数
        x = x - learning_rate * gradient
        print(f"迭代{i}: x = {x:.4f}")
    
    return x  # 接近 0

# 结果：x 会逐渐接近 0（最小值点）
```

## 四、概率统计：不确定性的语言

### 1. 概率基础

```
概率 = 事件发生的可能性

P(事件) = 有利情况数 / 总情况数

例子：
P(抛硬币正面) = 1/2 = 0.5
P(掷骰子得 6) = 1/6 ≈ 0.167
```

### 2. 常见概率分布

```
┌─────────────────────────────────────────────────────────────┐
│ 正态分布 (高斯分布) - 最常见的分布                           │
│                                                             │
│         ╱╲                                                  │
│        ╱  ╲                                                 │
│       ╱    ╲                                                │
│      ╱      ╲                                               │
│─────╱────────╲─────                                         │
│    μ-2σ μ  μ+2σ                                            │
│                                                             │
│  应用：身高、体重、测量误差、股票收益...                    │
└─────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────┐
│ 均匀分布 - 所有结果等可能                                     │
│                                                             │
│  ┌───────────────┐                                          │
│  │               │                                          │
│  │               │                                          │
│──┴───────────────┴──                                        │
│                                                             │
│  应用：随机数生成、抽样...                                  │
└─────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────┐
│ 二项分布 - n 次试验中成功 k 次的概率                           │
│                                                             │
│         ╱╲                                                  │
│        ╱  ╲                                                 │
│       ╱    ╲╲                                               │
│      ╱      ╲ ╲                                             │
│─────╱────────╲─╲───                                         │
│    0  1  2  3  4  5                                         │
│                                                             │
│  应用：抛硬币、质量检测、A/B 测试...                          │
└─────────────────────────────────────────────────────────────┘
```

### 3. 期望和方差

```
期望 (均值) = 长期平均结果
E(X) = Σ [x × P(x)]

例子：掷骰子
E(X) = 1×(1/6) + 2×(1/6) + ... + 6×(1/6) = 3.5

方差 = 数据的离散程度
Var(X) = E[(X - μ)²]

方差大：数据分散    方差小：数据集中
  ●   ● ●             ●●●●●
●       ●   ●           ●
```

### 4. 条件概率和贝叶斯定理

```
条件概率：P(A|B) = 在 B 发生的条件下 A 发生的概率

贝叶斯定理：
                P(B|A) × P(A)
P(A|B) = ─────────────────────
                      P(B)

直觉理解：
后验概率 = (似然 × 先验) / 证据

应用：
- 垃圾邮件过滤
- 医学诊断
- 更新信念
```

### 5. 贝叶斯例子：疾病检测

```
问题：检测呈阳性，真的患病的概率是多少？

已知：
- 疾病发病率：1% (P(病) = 0.01)
- 检测准确率：99% (P(阳性 | 病) = 0.99)
- 假阳性率：5% (P(阳性 | 无病) = 0.05)

计算：
P(病 | 阳性) = P(阳性 | 病) × P(病) / P(阳性)
            = 0.99 × 0.01 / (0.99×0.01 + 0.05×0.99)
            = 0.0099 / 0.0594
            ≈ 16.7%

反直觉！即使检测 99% 准确，阳性也不代表大概率患病
原因：疾病太罕见，假阳性比真阳性多
```

## 五、统计推断：从样本到总体

### 1. 总体 vs 样本

```
总体：所有可能的数据（通常无法获得）
样本：实际收集到的数据（总体的子集）

目标：用样本推断总体特征

例子：
总体：全国所有人的身高
样本：随机抽取的 1000 人身高
推断：全国平均身高 ≈ 样本平均身高
```

### 2. 中心极限定理

```
核心思想：大量独立随机变量的和近似服从正态分布

不管原始分布是什么形状：
  ┌─┐     ╱╲     ╱╲
  │ │    ╱  ╲   ╱  ╲
──┴─┴──  ╱    ╲ ╱    ╲
均匀   三角   奇怪   正态！
分布   分布   分布   分布

(抽样均值的分布)

应用：
- 置信区间
- 假设检验
- A/B 测试
```

### 3. 假设检验

```
步骤：
1. 提出假设
   - 零假设 H₀: 没有差异/效果
   - 备择假设 H₁: 有差异/效果

2. 收集数据，计算统计量

3. 计算 p 值
   p 值 = 在 H₀ 为真时，观察到当前结果的概率

4. 做决策
   p < 0.05 → 拒绝 H₀ (显著)
   p ≥ 0.05 → 不拒绝 H₀ (不显著)

可视化：
         正态分布曲线
            ╱╲
           ╱  ╲
          ╱    ╲
         ╱      ╲
────────╱────────╲────────
       ←  p 值  →
      (尾部面积)
```

## 六、信息论基础

### 1. 信息量

```
信息量 = 消除不确定性的程度

意外的事件 → 信息量大
常见的事件 → 信息量小

公式：I(x) = -log₂(P(x))

例子：
- "明天太阳从东边升起" → 信息量≈0 (确定)
- "明天下雪" (在热带) → 信息量大 (意外)
```

### 2. 熵 (Entropy)

```
熵 = 平均信息量 = 不确定性

H(X) = -Σ P(x) × log₂(P(x))

熵最大：均匀分布（最不确定）
熵最小：确定事件（最确定）

例子：
公平硬币：H = 1 bit (最大熵)
两面都是正面：H = 0 (无熵)
```

### 3. 交叉熵 (Cross Entropy)

```
交叉熵 = 用预测分布 Q 编码真实分布 P 的平均长度

H(P, Q) = -Σ P(x) × log₂(Q(x))

在机器学习中：
- P = 真实标签
- Q = 模型预测
- 交叉熵 = 损失函数

交叉熵越小，预测越准确！
```

## 七、数学在 ML 中的实际应用

### 线性回归的数学

```
模型：y = w·x + b

损失函数（均方误差）：
L(w,b) = (1/n) × Σ(yᵢ - (w·xᵢ + b))²

求最优参数：
∂L/∂w = 0
∂L/∂b = 0

解得：
w = Σ((xᵢ - x̄)(yᵢ - ȳ)) / Σ((xᵢ - x̄)²)
b = ȳ - w·x̄
```

### 逻辑回归的数学

```
模型：P(y=1|x) = σ(w·x + b)

其中 σ 是 sigmoid 函数：
σ(z) = 1 / (1 + e⁻ᶻ)

损失函数（交叉熵）：
L = -Σ[yᵢ·log(pᵢ) + (1-yᵢ)·log(1-pᵢ)]

用梯度下降优化 w 和 b
```

### 神经网络的数学

```
前向传播：
z = W·x + b
a = σ(z)  (激活函数)

反向传播：
∂L/∂W = ∂L/∂a × ∂a/∂z × ∂z/∂W
      = δ · xᵀ

用链式法则逐层传播梯度
```

## 八、学习建议

### 📚 学习顺序

```
1. 线性代数基础 (向量、矩阵、乘法)
        ↓
2. 微积分基础 (导数、梯度)
        ↓
3. 概率基础 (分布、期望、贝叶斯)
        ↓
4. 在学习算法时深入相关数学
```

### 💡 关键建议

1. **不要一开始就深究所有数学**
   - 先理解算法直觉
   - 遇到不懂的数学再回头学

2. **可视化帮助理解**
   - 画图理解导数、梯度
   - 用几何理解线性代数

3. **动手推导**
   - 自己推一遍公式
   - 写代码实现数学运算

4. **善用工具**
   - NumPy 处理矩阵
   - SymPy 符号计算
   - 可视化工具
