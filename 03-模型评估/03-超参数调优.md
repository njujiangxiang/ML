## 一、超参数 vs 参数

### 区别

```
参数 (Parameters):
- 模型内部学习得到
- 如：权重 w、偏置 b
- 训练过程自动优化

超参数 (Hyperparameters):
- 训练前手动设置
- 如：学习率、树的深度、K 值
- 需要人工调优

例子 (随机森林):
参数：
- 每个节点的分裂点
- 每个叶子的预测值

超参数：
- n_estimators (树的数量)
- max_depth (最大深度)
- min_samples_split (最小分裂样本数)
```

### 调参的重要性

```
好超参数 vs 差超参数：

差超参数：
- 模型欠拟合或过拟合
- 性能差
- 训练慢

好超参数：
- 模型性能最优
- 泛化能力强
- 训练效率高

调参目标：找到最佳超参数组合！
```

## 二、网格搜索 (Grid Search)

### 原理

```
网格搜索 = 穷举所有可能的组合

例子 (2 个参数):
n_estimators: [50, 100, 200]
max_depth: [None, 10, 20]

组合：
(50, None), (50, 10), (50, 20)
(100, None), (100, 10), (100, 20)
(200, None), (200, 10), (200, 20)

共 3×3 = 9 种组合，每种训练并评估
```

### 可视化

```
参数空间网格：

max_depth
  ↑
  │  ●  ●  ●  ← n_estimators=200
  │
  │  ●  ●  ●  ← n_estimators=100
  │
  │  ●  ●  ●  ← n_estimators=50
  └────────────→
     None 10  20

每个●都要训练一次！
```

### 代码实现

```python
from sklearn.model_selection import GridSearchCV
from sklearn.ensemble import RandomForestClassifier
from sklearn.datasets import load_iris

# 加载数据
iris = load_iris()
X, y = iris.data, iris.target

# 参数网格
param_grid = {
    'n_estimators': [50, 100, 200],
    'max_depth': [None, 10, 20],
    'min_samples_split': [2, 5, 10],
    'min_samples_leaf': [1, 2, 4]
}

# 基础模型
rf = RandomForestClassifier(random_state=42)

# 网格搜索
grid_search = GridSearchCV(
    estimator=rf,
    param_grid=param_grid,
    cv=5,                    # 5 折交叉验证
    scoring='accuracy',
    n_jobs=-1,               # 并行计算
    verbose=1,               # 输出进度
    return_train_score=True  # 返回训练分数
)

# 执行搜索
print("开始网格搜索...")
grid_search.fit(X, y)

# 结果
print(f"\n最佳参数：{grid_search.best_params_}")
print(f"最佳分数：{grid_search.best_score_:.4f}")
print(f"最佳模型：{grid_search.best_estimator_}")
```

## 三、随机搜索 (Random Search)

### 原理

```
随机搜索 = 随机采样参数组合

vs 网格搜索：
网格搜索：9 次试验 (3×3)，只覆盖网格点
随机搜索：9 次试验，随机覆盖整个空间

优势：
✓ 更高效 (Bergstra & Bengio, 2012)
✓ 可以探索更多值
✓ 适合参数多的情况
```

### 可视化

```
参数空间采样：

网格搜索：          随机搜索：
  ●  ●  ●            ●    ●
  ●  ●  ●              ●
  ●  ●  ●            ●    ●
(固定网格)        (随机分布)
```

### 代码实现

```python
from sklearn.model_selection import RandomizedSearchCV
from scipy.stats import randint, uniform

# 参数分布 (不是固定值)
param_dist = {
    'n_estimators': randint(50, 300),      # 50-300 随机整数
    'max_depth': [None] + list(range(10, 50, 5)),
    'min_samples_split': randint(2, 20),
    'min_samples_leaf': randint(1, 10),
    'max_features': ['sqrt', 'log2', None]
}

# 随机搜索
random_search = RandomizedSearchCV(
    estimator=RandomForestClassifier(random_state=42),
    param_distributions=param_dist,
    n_iter=50,               # 尝试 50 组参数
    cv=5,
    scoring='accuracy',
    n_jobs=-1,
    verbose=1,
    random_state=42
)

print("开始随机搜索...")
random_search.fit(X, y)

print(f"\n最佳参数：{random_search.best_params_}")
print(f"最佳分数：{random_search.best_score_:.4f}")
```

### 网格 vs 随机 对比

```python
import time

# 网格搜索
start = time.time()
grid_search = GridSearchCV(
    RandomForestClassifier(random_state=42),
    param_grid={'n_estimators': [50, 100, 200], 'max_depth': [None, 10, 20]},
    cv=5, n_jobs=-1
)
grid_search.fit(X, y)

# 随机搜索

    RandomForestClassifier(random_state=42),
    param_distributions={'n_estimators': randint(50, 200), 'max_depth': [None] + list(range(10, 30, 5))},
    n_iter=20, cv=5, n_jobs=-1, random_state=42
)
random_search.fit(X, y)
random_time = time.time() - start

print(f"网格搜索：{grid_time:.2f}秒，最佳分数：{grid_search.best_score_:.4f}")
print(f"随机搜索：{random_time:.2f}秒，最佳分数：{random_search.best_score_:.4f}")
```

## 四、贝叶斯优化

### 原理

```
贝叶斯优化 = 智能搜索

思想：
- 用之前的结果指导后续搜索
- 平衡"探索"和"利用"
- 更高效找到最优解

vs 网格/随机：
网格/随机：每次试验独立
贝叶斯：记住历史，智能选择下一个点
```

### 使用 Hyperopt

```python
from hyperopt import fmin, tpe, hp, STATUS_OK, Trials
from sklearn.ensemble import RandomForestClassifier
from sklearn.model_selection import cross_val_score
import numpy as np

# 定义参数空间
space = {
    'n_estimators': hp.quniform('n_estimators', 50, 300, 10),
    'max_depth': hp.choice('max_depth', [None] + list(range(10, 50, 5))),
    'min_samples_split': hp.quniform('min_samples_split', 2, 20, 1),
    'min_samples_leaf': hp.quniform('min_samples_leaf', 1, 10, 1)
}

# 目标函数
def objective(params):
    params['n_estimators'] = int(params['n_estimators'])
    params['min_samples_split'] = int(params['min_samples_split'])
    params['min_samples_leaf'] = int(params['min_samples_leaf'])
    
    model = RandomForestClassifier(**params, random_state=42)
    score = cross_val_score(model, X, y, cv=5, scoring='accuracy').mean()
    
    # 最小化负准确率
    return {'loss': -score, 'status': STATUS_OK}

# 执行优化
trials = Trials()
best = fmin(
    fn=objective,
    space=space,
    algo=tpe.suggest,
    max_evals=50,
    trials=trials
)

print(f"最佳参数：{best}")
```

### 使用 Optuna (推荐)

```python
import optuna
from optuna.visualization import plot_optimization_history, plot_param_importances

def objective(trial):
    params = {
        'n_estimators': trial.suggest_int('n_estimators', 50, 300),
        'max_depth': trial.suggest_categorical('max_depth', [None, 10, 20, 30, 40]),
        'min_samples_split': trial.suggest_int('min_samples_split', 2, 20),
        'min_samples_leaf': trial.suggest_int('min_samples_leaf', 1, 10),
        'max_features': trial.suggest_categorical('max_features', ['sqrt', 'log2', None])
    }
    
    model = RandomForestClassifier(**params, random_state=42)
    score = cross_val_score(model, X, y, cv=5, scoring='accuracy').mean()
    
    return score

# 创建研究
study = optuna.create_study(direction='maximize')

# 优化
study.optimize(objective, n_trials=50, show_progress_bar=True)

# 结果
print(f"\n最佳参数：{study.best_params}")
print(f"最佳分数：{study.best_value:.4f}")

# 可视化
fig1 = plot_optimization_history(study)
fig1.write_image('../images/超参数调优 - 优化历史.png')

fig2 = plot_param_importances(study)
fig2.write_image('../images/超参数调优 - 参数重要性.png')
```

## 五、自动化调参管道

### 完整示例

```python
from sklearn.pipeline import Pipeline
from sklearn.preprocessing import StandardScaler
from sklearn.decomposition import PCA
from sklearn.ensemble import RandomForestClassifier
from sklearn.model_selection import GridSearchCV

# 创建管道
pipeline = Pipeline([
    ('scaler', StandardScaler()),
    ('pca', PCA()),
    ('clf', RandomForestClassifier())
])

# 参数网格 (包括预处理)
param_grid = {
    'pca__n_components': [None, 10, 20, 30],
    'clf__n_estimators': [100, 200],
    'clf__max_depth': [None, 10, 20],
    'clf__min_samples_split': [2, 5]
}

# 网格搜索
grid_search = GridSearchCV(
    pipeline,
    param_grid,
    cv=5,
    scoring='accuracy',
    n_jobs=-1,
    verbose=1
)

grid_search.fit(X, y)

print(f"最佳参数：{grid_search.best_params_}")
print(f"最佳分数：{grid_search.best_score_:.4f}")

# 使用最佳模型预测
best_model = grid_search.best_estimator_
```

## 六、调参最佳实践

### 建议

```
1. 先粗后细
   第一次：大范围，少参数
   第二次：小范围，多参数

2. 参数优先级
   高影响：学习率、树深度、正则化
   低影响：迭代次数、批量大小

3. 使用对数尺度
   学习率：[0.0001, 0.001, 0.01, 0.1]
   不是：[0.01, 0.02, 0.03, ...]

4. 早停策略
   训练过程中自动停止
   节省时间

5. 并行计算
   n_jobs=-1 使用所有 CPU
```

### 常见陷阱

```
✗ 在测试集上调参
  → 数据泄露！

✗ 参数范围太小
  → 错过最优解

✗ 参数范围太大
  → 浪费时间

✗ 忽略默认值
  → 默认值可能已经很好

✗ 过度调参
  → 在验证集上过拟合
```
