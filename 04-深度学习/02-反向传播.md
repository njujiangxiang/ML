## 一、直观理解反向传播

### 核心思想

```
反向传播 = 误差从后往前传，告诉每个权重"你有多大责任"

类比：
老板批评项目失败
  ↓
经理责任 70%，员工责任 30%
  ↓
经理怪供应商，员工怪队友...

每个环节都计算：
- 我有多大责任？(梯度)
- 我该怎么改？(更新权重)
```

### 前向 vs 反向

```
前向传播 (计算预测):
输入 → 隐藏层 → 输出层 → 预测 → 计算损失

    x ──→ [h] ──→ [y] ──→ loss
          │       │
          W₁      W₂

反向传播 (计算梯度):
损失 → 输出层 → 隐藏层 → 输入层 → 更新权重

    loss ←─ [y] ←─ [h] ←─ x
           │       │
          ∂W₂     ∂W₁
```

## 二、链式法则：反向传播的数学基础

### 链式法则回顾

```
复合函数求导：

y = f(u), u = g(x)

dy/dx = dy/du × du/dx

可视化：
x ──→ u ──→ y
      │     │
     du/dx dy/du
     
     dy/dx = dy/du × du/dx
```

### 神经网络中的应用

```
简单网络：
x ──→ z₁ = W₁x + b₁ ──→ h = σ(z₁) ──→ z₂ = W₂h + b₂ ──→ ŷ = σ(z₂)

损失 L 对 W₁ 的导数：

∂L/∂W₁ = ∂L/∂ŷ × ∂ŷ/∂z₂ × ∂z₂/∂h × ∂h/∂z₁ × ∂z₁/∂W₁
       ↑         ↑         ↑         ↑         ↑
    损失梯度  输出层    权重     激活     权重

从后往前乘！
```

## 三、完整推导：单隐藏层网络

### 网络结构

```
网络：
- 输入层：x (维度：n_x)
- 隐藏层：h = σ(W₁x + b₁) (维度：n_h)
- 输出层：ŷ = σ(W₂h + b₂) (维度：1)

损失 (二分类交叉熵):
L = -[y·log(ŷ) + (1-y)·log(1-ŷ)]
```

### 符号说明

```
前向传播：
z₁ = W₁x + b₁          (隐藏层加权输入)
h = σ(z₁)              (隐藏层激活)
z₂ = W₂h + b₂          (输出层加权输入)
ŷ = σ(z₂)              (输出预测)

损失：
L = -[y·log(ŷ) + (1-y)·log(1-ŷ)]
```

### 反向传播推导

```
步骤 1: 输出层误差 δ₂

δ₂ = ∂L/∂z₂
   = ∂L/∂ŷ × ∂ŷ/∂z₂
   
∂L/∂ŷ = -y/ŷ + (1-y)/(1-ŷ)
∂ŷ/∂z₂ = ŷ(1-ŷ)  (sigmoid 导数)

δ₂ = (-y/ŷ + (1-y)/(1-ŷ)) × ŷ(1-ŷ)
   = -y(1-ŷ) + (1-y)ŷ
   = ŷ - y

惊喜！输出层误差 = 预测 - 真实
```

```
步骤 2: 输出层权重梯度

∂L/∂W₂ = δ₂ × ∂z₂/∂W₂
       = δ₂ × h

∂L/∂b₂ = δ₂ × ∂z₂/∂b₂
       = δ₂
```

```
步骤 3: 隐藏层误差 δ₁

δ₁ = ∂L/∂z₁
   = (∂L/∂z₂ × ∂z₂/∂h) × ∂h/∂z₁
   = (δ₂ × W₂) × σ'(z₁)
   = (δ₂ × W₂) × h(1-h)

误差从输出层传回隐藏层！
```

```
步骤 4: 隐藏层权重梯度

∂L/∂W₁ = δ₁ × ∂z₁/∂W₁
       = δ₁ × x

∂L/∂b₁ = δ₁ × ∂z₁/∂b₁
       = δ₁
```

### 更新权重

```
梯度下降更新：

W₂ = W₂ - η × ∂L/∂W₂
b₂ = b₂ - η × ∂L/∂b₂
W₁ = W₁ - η × ∂L/∂W₁
b₁ = b₁ - η × ∂L/∂b₁

其中 η 是学习率
```

## 四、代码实现：从零开始

```python
import numpy as np
import matplotlib.pyplot as plt

class NeuralNetworkBP:
    """带反向传播的神经网络"""
    
    def __init__(self, layer_sizes):
        """
        layer_sizes: 每层神经元数，如 [2, 4, 1]
        """
        self.layer_sizes = layer_sizes
        self.n_layers = len(layer_sizes)
        
        # 初始化权重和偏置 (Xavier 初始化)
        self.weights = []
        self.biases = []
        
        for i in range(self.n_layers - 1):
            w = np.random.randn(layer_sizes[i], layer_sizes[i+1]) * \
                np.sqrt(2.0 / layer_sizes[i])
            b = np.zeros((1, layer_sizes[i+1]))
            self.weights.append(w)
            self.biases.append(b)
    
    def sigmoid(self, x):
        return 1 / (1 + np.exp(-np.clip(x, -500, 500)))
    
    def sigmoid_derivative(self, x):
        s = self.sigmoid(x)
        return s * (1 - s)
    
    def forward(self, X):
        """
        前向传播
        存储中间结果用于反向传播
        """
        self.activations = [X]  # a⁰, a¹, a², ...
        self.z_values = []      # z¹, z², ...
        
        current = X
        for i in range(self.n_layers - 1):
            z = np.dot(current, self.weights[i]) + self.biases[i]
            self.z_values.append(z)
            
            # 输出层用 sigmoid，隐藏层也用 sigmoid (方便演示)
            current = self.sigmoid(z)
            self.activations.append(current)
        
        return current  # 输出 ŷ
    
    def backward(self, X, y, learning_rate=0.1):
        """
        反向传播
        计算梯度并更新权重
        """
        m = X.shape[0]  # 样本数
        
        # ===== 反向传播 =====
        
        # 1. 输出层误差 (交叉熵 + sigmoid 的简化)
        delta = self.activations[-1] - y.reshape(-1, 1)
        
        # 2. 从后向前传播
        gradients_w = []
        gradients_b = []
        
        for i in range(self.n_layers - 1, 0, -1):
            # 计算当前层梯度
            dW = np.dot(self.activations[i-1].T, delta) / m
            db = np.mean(delta, axis=0, keepdims=True)
            
            gradients_w.insert(0, dW)
            gradients_b.insert(0, db)
            
            # 传播误差到前一层 (如果不是第一层)
            if i > 1:
                delta = np.dot(delta, self.weights[i-1].T) * \
                        self.sigmoid_derivative(self.z_values[i-2])
        
        # ===== 更新权重 =====
        for i in range(self.n_layers - 1):
            self.weights[i] -= learning_rate * gradients_w[i]
            self.biases[i] -= learning_rate * gradients_b[i]
    
    def compute_loss(self, y_true, y_pred):
        """计算交叉熵损失"""
        epsilon = 1e-15
        y_pred = np.clip(y_pred, epsilon, 1 - epsilon)
        return -np.mean(y_true * np.log(y_pred) + 
                       (1 - y_true) * np.log(1 - y_pred))
    
    def fit(self, X, y, epochs=1000, learning_rate=0.1, verbose=True):
        """训练神经网络"""
        loss_history = []
        
        for epoch in range(epochs):
            # 前向传播
            y_pred = self.forward(X)
            
            # 计算损失
            loss = self.compute_loss(y, y_pred)
            loss_history.append(loss)
            
            # 反向传播 + 更新
            self.backward(X, y, learning_rate)
            
            if verbose and epoch % 100 == 0:
                print(f"Epoch {epoch:4d}, Loss: {loss:.6f}")
        
        return loss_history
    
    def predict(self, X):
        """预测类别"""
        proba = self.forward(X)
        return (proba >= 0.5).astype(int)
    
    def score(self, X, y):
        """计算准确率"""
        predictions = self.predict(X)
        return np.mean(predictions.flatten() == y.flatten())


# 测试：XOR 问题
if __name__ == "__main__":
    # XOR 数据
    X = np.array([[0, 0], [0, 1], [1, 0], [1, 1]])
    y = np.array([0, 1, 1, 0])  # XOR
    
    print("=== XOR 问题 ===")
    print("输入:")

    print("\n期望输出:")

    # 可视化
```

## 五、多层网络的反向传播

### 通用公式

```
对于 L 层网络：

输出层误差：
δᴸ = ∂L/∂zᴸ = ŷ - y  (交叉熵 + sigmoid)

隐藏层误差 (从后向前):
δˡ = (Wˡ)ᵀ × δˡ⁺¹ × σ'(zˡ)

权重梯度：
∂L/∂Wˡ = (aˡ⁻¹)ᵀ × δˡ / m
∂L/∂bˡ = mean(δˡ) / m

更新：
Wˡ = Wˡ - η × ∂L/∂Wˡ
bˡ = bˡ - η × ∂L/∂bˡ
```

### 代码实现 (任意层数)

```python
class DeepNeuralNetwork:
    """深层神经网络"""
    
    def __init__(self, layer_sizes, activation='relu'):
        self.layer_sizes = layer_sizes
        self.n_layers = len(layer_sizes)
        self.activation = activation
        
        # 初始化
        self.weights = []
        self.biases = []
        
        for i in range(self.n_layers - 1):
            w = np.random.randn(layer_sizes[i], layer_sizes[i+1]) * \
                np.sqrt(2.0 / layer_sizes[i])
            b = np.zeros((1, layer_sizes[i+1]))
            self.weights.append(w)
            self.biases.append(b)
    
    def relu(self, x):
        return np.maximum(0, x)
    
    def relu_derivative(self, x):
        return (x > 0).astype(float)
    
    def sigmoid(self, x):
        return 1 / (1 + np.exp(-np.clip(x, -500, 500)))
    
    def sigmoid_derivative(self, x):
        s = self.sigmoid(x)
        return s * (1 - s)
    
    def activate(self, x):
        if self.activation == 'relu':
            return self.relu(x)
        else:
            return self.sigmoid(x)
    
    def activate_derivative(self, x):
        if self.activation == 'relu':
            return self.relu_derivative(x)
        else:
            return self.sigmoid_derivative(x)
    
    def forward(self, X):
        self.activations = [X]
        self.z_values = []
        
        current = X
        for i in range(self.n_layers - 1):
            z = np.dot(current, self.weights[i]) + self.biases[i]
            self.z_values.append(z)
            
            # 最后一层用 sigmoid，其他用指定激活
            if i == self.n_layers - 2:
                current = self.sigmoid(z)
            else:
                current = self.activate(z)
            
            self.activations.append(current)
        
        return current
    
    def backward(self, X, y, learning_rate=0.01):
        m = X.shape[0]
        
        # 输出层误差 (交叉熵 + sigmoid)
        delta = self.activations[-1] - y.reshape(-1, 1)
        
        for i in range(self.n_layers - 1, 0, -1):
            # 梯度
            dW = np.dot(self.activations[i-1].T, delta) / m
            db = np.mean(delta, axis=0, keepdims=True)
            
            # 更新
            self.weights[i-1] -= learning_rate * dW
            self.biases[i-1] -= learning_rate * db
            
            # 传播误差 (如果不是第一层)
            if i > 1:
                delta = np.dot(delta, self.weights[i-2].T) * \
                        self.activate_derivative(self.z_values[i-2])
    
    def fit(self, X, y, epochs=1000, learning_rate=0.01):
        loss_history = []
        
        for epoch in range(epochs):
            output = self.forward(X)
            
            # 交叉熵损失
            epsilon = 1e-15
            output = np.clip(output, epsilon, 1 - epsilon)
            loss = -np.mean(y * np.log(output) + (1-y) * np.log(1-output))
            loss_history.append(loss)
            
            self.backward(X, y, learning_rate)
            
            if epoch % 100 == 0:
                print(f"Epoch {epoch}, Loss: {loss:.6f}")
        
        return loss_history
    
    def predict(self, X):
        return (self.forward(X) >= 0.5).astype(int)


# 测试：多层网络
if __name__ == "__main__":
    from sklearn.datasets import make_moons
    
    # 生成月牙形数据 (非线性可分)
    X, y = make_moons(n_samples=1000, noise=0.2, random_state=42)
    
    # 创建深层网络
    nn = DeepNeuralNetwork([2, 16, 8, 1], activation='relu')
    
    # 训练
    print("训练深层网络...")
    loss_history = nn.fit(X, y, epochs=2000, learning_rate=0.1)
    
    # 评估
    print(f"\n训练准确率：{nn.score(X, y):.4f}")
    
    # 可视化决策边界
    import matplotlib.pyplot as plt
    
    x_min, x_max = X[:, 0].min() - 0.5, X[:, 0].max() + 0.5
    y_min, y_max = X[:, 1].min() - 0.5, X[:, 1].max() + 0.5
    xx, yy = np.meshgrid(np.linspace(x_min, x_max, 200),
                         np.linspace(y_min, y_max, 200))
    
    Z = nn.forward(np.c_[xx.ravel(), yy.ravel()])
    Z = Z.reshape(xx.shape)
    
    plt.figure(figsize=(10, 8))
    plt.contourf(xx, yy, Z, levels=50, cmap='RdYlBu', alpha=0.6)
    plt.scatter(X[:, 0], X[:, 1], c=y, cmap='RdYlBu', 
               s=50, edgecolors='k', alpha=0.7)
    plt.xlabel('x₁')
    plt.ylabel('x₂')
```

## 六、梯度消失和梯度爆炸

### 问题描述

```
梯度消失：
深层网络中，梯度从后向前传，越来越小

δˡ = (Wˡ)ᵀ × δˡ⁺¹ × σ'(zˡ)
      ↑         ↑        ↑
    权重     后层梯度   激活导数

如果每项都 < 1:
0.5 × 0.5 × 0.5 × ... → 0

前面层的权重几乎不更新！
```

```
梯度爆炸：
如果每项都 > 1:
2 × 2 × 2 × ... → ∞

权重更新过大，网络不稳定！
```

### 可视化

```
梯度大小 vs 层数：

梯度
 ↑
 │╲  梯度爆炸
 │ ╲
 │  ╲
 │   ╲
 │    ─────  正常
 │   ╱
 │  ╱
 │ ╱  梯度消失
 └────────────→ 层数
 1  2  3  4  5
```

### 解决方案

```
1. 激活函数选择
   ✗ Sigmoid/Tanh: 导数最大 0.25，容易消失
   ✓ ReLU: 导数为 1 (正区间)，缓解消失

2. 权重初始化
   ✗ 随机初始化：可能太大或太小
   ✓ Xavier 初始化：w ~ N(0, √(2/n_in))
   ✓ He 初始化 (ReLU): w ~ N(0, √(2/n_in))

3. Batch Normalization
   每层标准化，加速训练

4. 残差连接 (ResNet)
   跳跃连接，梯度直接传回

5. 梯度裁剪 (针对爆炸)
   if ||gradient|| > threshold:
       gradient = gradient * threshold / ||gradient||
```

### 代码：梯度裁剪

```python
def clip_gradients(gradients, max_norm=5.0):
    """梯度裁剪"""
    total_norm = np.sqrt(sum(np.sum(g**2) for g in gradients))
    
    if total_norm > max_norm:
        scale = max_norm / total_norm
        gradients = [g * scale for g in gradients]
    
    return gradients

# 在反向传播中使用
gradients = [dW1, dW2, dW3, ...]
clipped_gradients = clip_gradients(gradients, max_norm=5.0)
```
