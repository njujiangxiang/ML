```
生物神经元结构：

      │
      ↓
```

```
人工神经元 (感知机):

    x₁ ─────┐
            │
    x₂ ─────┼──→ Σ ─→ 激活函数 ─→ y
            │     (加权求和)
    x₃ ─────┘

    偏置：b

数学表达：
z = w₁x₁ + w₂x₂ + w₃x₃ + b
y = f(z)  (f 是激活函数)
```

## 二、感知机：最早的神经网络

### 感知机原理

```
感知机 (1957 年，Frank Rosenblatt):

输入：x = [x₁, x₂, ..., xₙ]
权重：w = [w₁, w₂, ..., wₙ]
偏置：b

计算：
z = w·x + b
y = step(z)  (阶跃函数)

阶跃函数：
     │
  1 ─┼──────╱────
     │    ╱
     │   ╱
  0 ─┼──╱───────
     └──────────→ z
         0

z ≥ 0 → y = 1
z < 0 → y = 0
```

### 感知机学习规则

```
感知机学习算法：

1. 初始化权重 w 和偏置 b (随机或 0)

2. 对每个训练样本 (x, y):
   a. 预测：ŷ = step(w·x + b)
   b. 更新权重：
      wᵢ = wᵢ + η × (y - ŷ) × xᵢ
      b = b + η × (y - ŷ)
   
   其中 η 是学习率

3. 重复直到收敛

直觉：
- 预测对了：不更新
- 预测错了：往正确方向调整
```

### 代码实现

```python
import numpy as np

class Perceptron:
    def __init__(self, learning_rate=0.01, n_iterations=100):
        self.lr = learning_rate
        self.n_iterations = n_iterations
        self.weights = None
        self.bias = None
    
    def step_function(self, z):
        """阶跃函数"""
        return np.where(z >= 0, 1, 0)
    
    def fit(self, X, y):
        """训练感知机"""
        n_samples, n_features = X.shape
        
        # 初始化
        self.weights = np.zeros(n_features)
        self.bias = 0
        
        # 迭代训练
        for iteration in range(self.n_iterations):
            for idx, x_i in enumerate(X):
                # 线性组合
                z = np.dot(x_i, self.weights) + self.bias
                
                # 预测
                y_pred = self.step_function(z)
                
                # 更新权重
                error = y[idx] - y_pred
                self.weights += self.lr * error * x_i
                self.bias += self.lr * error
        
        return self
    
    def predict(self, X):
        """预测"""
        z = np.dot(X, self.weights) + self.bias
        return self.step_function(z)


# 测试：与门 (AND gate)
if __name__ == "__main__":
    # AND 门真值表
    X = np.array([
        [0, 0],
        [0, 1],
        [1, 0],
        [1, 1]
    ])
    y = np.array([0, 0, 0, 1])  # AND 输出
    
    perceptron = Perceptron(learning_rate=0.1, n_iterations=50)
    perceptron.fit(X, y)
    
    print("AND 门测试结果:")
    for x, true_y in zip(X, y):
        pred = perceptron.predict(x.reshape(1, -1))[0]
        print(f"  {x[0]} AND {x[1]} = {pred} (真实：{true_y})")
    
    print(f"\n学到的权重：w = {perceptron.weights}")
    print(f"学到的偏置：b = {perceptron.bias:.4f}")
```

### 感知机的局限

```
感知机只能解决线性可分问题！

线性可分 (AND 门):
  x₂
  ↑
  │  ○ ●
  │○ │ ●
  ──┼────→ 可以用直线分开
  │  ●
  └────→ x₁

线性不可分 (XOR 门):
  x₂
  ↑
  │  ● ○
  │○ │ ●
  ──┼────→ 无法用直线分开！
  │  ● ○
  └────→ x₁

1969 年，Minsky & Papert 证明了这个局限
→ 神经网络研究进入寒冬
```

## 三、多层神经网络：突破局限

### 网络结构

```
多层感知机 (MLP):

  输入层    隐藏层    输出层
  
    x₁       h₁       y₁
    x₂       h₂       y₂
    x₃       h₃
    ...      ...
    
层间全连接：
- 每个输入连接到每个隐藏神经元
- 每个隐藏神经元连接到每个输出神经元

为什么多层更强？
✓ 可以学习非线性决策边界
✓ 可以解决 XOR 问题
✓ 可以逼近任意函数 (通用近似定理)
```

### 前向传播

```
数据从输入流向输出：

输入层 → 隐藏层 → 输出层

计算过程：
1. 输入层：接收数据 x
2. 隐藏层：z₁ = W₁·x + b₁
           h = f(z₁)  (激活)
3. 输出层：z₂ = W₂·h + b₂
           ŷ = g(z₂)  (激活)

符号：
W₁: 输入到隐藏的权重矩阵
b₁: 隐藏层偏置
f: 隐藏层激活函数
W₂: 隐藏到输出的权重矩阵
b₂: 输出层偏置
g: 输出层激活函数
```

### 代码实现

```python
class NeuralNetwork:
    def __init__(self, layer_sizes):
        """
        layer_sizes: 每层的神经元数量
        例如：[2, 4, 1] 表示 2 输入，4 隐藏，1 输出
        """
        self.layer_sizes = layer_sizes
        self.weights = []
        self.biases = []
        
        # 初始化权重和偏置
        for i in range(len(layer_sizes) - 1):
            # Xavier 初始化
            w = np.random.randn(layer_sizes[i], layer_sizes[i+1]) * \
                np.sqrt(2.0 / layer_sizes[i])
            b = np.zeros((1, layer_sizes[i+1]))
            self.weights.append(w)
            self.biases.append(b)
    
    def sigmoid(self, x):
        return 1 / (1 + np.exp(-np.clip(x, -500, 500)))
    
    def sigmoid_derivative(self, x):
        s = self.sigmoid(x)
        return s * (1 - s)
    
    def relu(self, x):
        return np.maximum(0, x)
    
    def relu_derivative(self, x):
        return (x > 0).astype(float)
    
    def forward(self, X):
        """前向传播"""
        self.activations = [X]  # 存储每层的激活值
        self.z_values = []      # 存储每层的加权输入
        
        current = X
        for i in range(len(self.weights)):
            # 加权求和
            z = np.dot(current, self.weights[i]) + self.biases[i]
            self.z_values.append(z)
            
            # 激活 (输出层用 sigmoid，隐藏层用 ReLU)
            if i == len(self.weights) - 1:
                current = self.sigmoid(z)  # 输出层
            else:
                current = self.relu(z)     # 隐藏层
            
            self.activations.append(current)
        
        return current
    
    def backward(self, X, y, learning_rate=0.01):
        """反向传播"""
        m = X.shape[0]
        
        # 输出层误差
        delta = self.activations[-1] - y.reshape(-1, 1)
        
        # 从后向前传播误差
        for i in range(len(self.weights) - 1, -1, -1):
            # 计算梯度
            dW = np.dot(self.activations[i].T, delta) / m
            db = np.mean(delta, axis=0, keepdims=True)
            
            # 更新权重
            self.weights[i] -= learning_rate * dW
            self.biases[i] -= learning_rate * db
            
            # 传播误差到前一层
            if i > 0:
                delta = np.dot(delta, self.weights[i].T) * \
                        self.relu_derivative(self.z_values[i-1])
    
    def fit(self, X, y, epochs=1000, learning_rate=0.01, verbose=True):
        """训练神经网络"""
        loss_history = []
        
        for epoch in range(epochs):
            # 前向传播
            output = self.forward(X)
            
            # 计算损失 (MSE)
            loss = np.mean((output - y.reshape(-1, 1)) ** 2)
            loss_history.append(loss)
            
            # 反向传播
            self.backward(X, y, learning_rate)
            
            if verbose and epoch % 100 == 0:
                print(f"Epoch {epoch}, Loss: {loss:.6f}")
        
        return loss_history
    
    def predict(self, X):
        """预测"""
        output = self.forward(X)
        return (output >= 0.5).astype(int)


# 测试：XOR 门
if __name__ == "__main__":
    # XOR 真值表
    X = np.array([[0, 0], [0, 1], [1, 0], [1, 1]])
    y = np.array([0, 1, 1, 0])  # XOR 输出
    
    # 创建网络：2 输入，4 隐藏，1 输出
    nn = NeuralNetwork([2, 4, 1])
    
    # 训练
    print("训练神经网络...")

    # 测试

    plt.show()
```

## 四、激活函数：引入非线性

### 为什么需要激活函数？

```
没有激活函数：
  多层网络 = 单层网络！
  
  y = W₂(W₁x + b₁) + b₂
    = (W₂W₁)x + (W₂b₁ + b₂)
    = W'x + b'
  
  还是线性！白搭了多层。

有激活函数：
  y = f(W₂·f(W₁x + b₁) + b₂)
  
  非线性！可以学习复杂模式。
```

### 常见激活函数

```
1. Sigmoid
   f(x) = 1 / (1 + e⁻ˣ)
   
        │
     1 ─┼──────╱────
        │    ╱
   0.5 ─┼───●──────
        │ ╱
     0 ─┼╱────────
        └──────────→ x
       -3   0   3
   
   优点：输出 (0,1)，可解释为概率
   缺点：梯度消失，输出不以 0 为中心

2. Tanh (双曲正切)
   f(x) = tanh(x)
   
        │
     1 ─┼────╱────
        │  ╱
     0 ─┼─●───────
        │╱
    -1 ─┼────────
        └──────────→ x
       -3   0   3
   
   优点：输出 (-1,1)，以 0 为中心
   缺点：梯度消失

3. ReLU (修正线性单元)
   f(x) = max(0, x)
   
        │╱
        │╱
        │╱
     0 ─┼───────
        │
        └──────────→ x
       -3   0   3
   
   优点：计算快，梯度不消失
   缺点：Dead ReLU (负输入梯度为 0)

4. Leaky ReLU
   f(x) = max(αx, x), α≈0.01
   
        │╱
        │╱
     0 ─┼───────
       ╱│
      ╱ │
        └──────────→ x
   
   优点：解决 Dead ReLU
   缺点：需要选择α

5. Softmax (多分类输出层)
   f(xᵢ) = eˣⁱ / Σⱼeˣʲ
   
   输出概率分布，和为 1
```

### 选择激活函数

```
推荐：
- 隐藏层：ReLU (默认) 或 Leaky ReLU
- 二分类输出：Sigmoid
- 多分类输出：Softmax
- 回归输出：Linear (无激活)
```

## 五、损失函数：衡量预测好坏

### 回归问题

```
均方误差 (MSE):
L = (1/n) × Σ(yᵢ - ŷᵢ)²

特点：
✓ 对大误差敏感
✓ 处处可导
✓ 最常用

平均绝对误差 (MAE):
L = (1/n) × Σ|yᵢ - ŷᵢ|

特点：
✓ 对异常值鲁棒
✗ 在 0 点不可导
```

### 分类问题

```
交叉熵损失 (Cross Entropy):

二分类:
L = -[y·log(ŷ) + (1-y)·log(1-ŷ)]

多分类:
L = -Σ yᵢ·log(ŷᵢ)

其中：
- y: 真实标签 (one-hot)
- ŷ: 预测概率

特点：
✓ 梯度大，学习快
✓ 适合概率输出
✓ 分类问题首选
```

### 代码实现

```python
def mse_loss(y_true, y_pred):
    """均方误差"""
    return np.mean((y_true - y_pred) ** 2)

def mae_loss(y_true, y_pred):
    """平均绝对误差"""
    return np.mean(np.abs(y_true - y_pred))

def binary_cross_entropy(y_true, y_pred):
    """二分类交叉熵"""
    epsilon = 1e-15
    y_pred = np.clip(y_pred, epsilon, 1 - epsilon)
    return -np.mean(y_true * np.log(y_pred) + 
                   (1 - y_true) * np.log(1 - y_pred))

def categorical_cross_entropy(y_true, y_pred):
    """多分类交叉熵"""
    epsilon = 1e-15
    y_pred = np.clip(y_pred, epsilon, 1 - epsilon)
    return -np.sum(y_true * np.log(y_pred)) / len(y_true)
```

## 六、优化器：更快更好地训练

### SGD (随机梯度下降)

```
基本 SGD:
w = w - η × ∇L

特点：
✓ 简单
✗ 可能震荡
✗ 学习率难调
```

### Momentum (动量)

```
SGD with Momentum:
v = γv + η∇L
w = w - v

其中 γ ≈ 0.9 (动量系数)

直觉：
- 下山时积累速度
- 减少震荡
- 加速收敛

可视化：
     ╱│╲
    ╱ │ ╲
   ╱  ↓  ╲  ← 有动量，冲过山谷
  ╱   │   ╲
 ╱────┴────╲
```

### Adam (自适应矩估计)

```
Adam (最常用):
- 结合 Momentum 和自适应学习率
- 每个参数有自己的学习率
- 默认推荐

参数：
- learning_rate: 0.001 (默认)
- beta1: 0.9 (一阶矩)
- beta2: 0.999 (二阶矩)
- epsilon: 1e-8 (数值稳定)
```

### 代码实现

```python
class AdamOptimizer:
    def __init__(self, learning_rate=0.001, beta1=0.9, beta2=0.999, epsilon=1e-8):
        self.lr = learning_rate
        self.beta1 = beta1
        self.beta2 = beta2
        self.epsilon = epsilon
        self.m = None  # 一阶矩
        self.v = None  # 二阶矩
        self.t = 0     # 时间步
    
    def update(self, params, grads):
        """更新参数"""
        if self.m is None:
            self.m = [np.zeros_like(p) for p in params]
            self.v = [np.zeros_like(p) for p in params]
        
        self.t += 1
        
        for i in range(len(params)):
            # 更新矩估计
            self.m[i] = self.beta1 * self.m[i] + (1 - self.beta1) * grads[i]
            self.v[i] = self.beta2 * self.v[i] + (1 - self.beta2) * (grads[i] ** 2)
            
            # 偏差修正
            m_hat = self.m[i] / (1 - self.beta1 ** self.t)
            v_hat = self.v[i] / (1 - self.beta2 ** self.t)
            
            # 更新参数
            params[i] -= self.lr * m_hat / (np.sqrt(v_hat) + self.epsilon)
        
        return params
```

```python
y_train_onehot = one_hot_encode(y_train, 10)
y_test_onehot = one_hot_encode(y_test, 10)

# 5. 创建神经网络
class NeuralNetworkMultiClass:
    def __init__(self, layer_sizes):
        self.layer_sizes = layer_sizes
        self.weights = []
        self.biases = []
        
        for i in range(len(layer_sizes) - 1):
            w = np.random.randn(layer_sizes[i], layer_sizes[i+1]) * \
                np.sqrt(2.0 / layer_sizes[i])
            b = np.zeros((1, layer_sizes[i+1]))
            self.weights.append(w)
            self.biases.append(b)
    
    def relu(self, x):
        return np.maximum(0, x)
    
    def relu_derivative(self, x):
        return (x > 0).astype(float)
    
    def softmax(self, x):
        exp_x = np.exp(x - np.max(x, axis=1, keepdims=True))
        return exp_x / np.sum(exp_x, axis=1, keepdims=True)
    
    def forward(self, X):
        self.activations = [X]
        self.z_values = []
        
        current = X
        for i in range(len(self.weights) - 1):
            z = np.dot(current, self.weights[i]) + self.biases[i]
            self.z_values.append(z)
            current = self.relu(z)
            self.activations.append(current)
        
        # 输出层用 softmax
        z = np.dot(current, self.weights[-1]) + self.biases[-1]
        self.z_values.append(z)
        current = self.softmax(z)
        self.activations.append(current)
        
        return current
    
    def backward(self, X, y, learning_rate=0.01):
        m = X.shape[0]
        
        # 输出层误差 (softmax + 交叉熵的导数)
        delta = self.activations[-1] - y
        
        for i in range(len(self.weights) - 1, -1, -1):
            dW = np.dot(self.activations[i].T, delta) / m
            db = np.mean(delta, axis=0, keepdims=True)
            
            self.weights[i] -= learning_rate * dW
            self.biases[i] -= learning_rate * db
            
            if i > 0:
                delta = np.dot(delta, self.weights[i].T) * \
                        self.relu_derivative(self.z_values[i-1])
    
    def fit(self, X, y, epochs=1000, learning_rate=0.01):
        loss_history = []
        
        for epoch in range(epochs):
            output = self.forward(X)
            
            # 交叉熵损失
            loss = -np.mean(np.sum(y * np.log(output + 1e-15), axis=1))
            loss_history.append(loss)
            
            self.backward(X, y, learning_rate)
            
            if epoch % 100 == 0:
                print(f"Epoch {epoch}, Loss: {loss:.4f}")
        
        return loss_history
    
    def predict(self, X):
        output = self.forward(X)
        return np.argmax(output, axis=1)
    
    def score(self, X, y):
        predictions = self.predict(X)
        return np.mean(predictions == np.argmax(y, axis=1))

# 6. 训练

print(f"测试准确率：{test_accuracy:.4f}")

# 8. 可视化
fig, axes = plt.subplots(1, 3, figsize=(15, 4))

# 损失曲线
axes[0].plot(loss_history)
axes[0].set_xlabel('Epoch')
axes[0].set_ylabel('Loss')
axes[0].set_title('训练损失曲线')
axes[0].grid(True, alpha=0.3)

# 预测 vs 真实
y_pred = nn.predict(X_test_scaled)
y_true = np.argmax(y_test_onehot, axis=1)

axes[1].scatter(y_true, y_pred, alpha=0.5)
axes[1].plot([0, 9], [0, 9], 'r--')
axes[1].set_xlabel('真实标签')
axes[1].set_ylabel('预测标签')
axes[1].set_title('预测 vs 真实')
axes[1].grid(True, alpha=0.3)

# 错误样本
errors = y_pred != y_true
error_indices = np.where(errors)[0][:5]

for i, idx in enumerate(error_indices):
    axes[2].imshow(X_test[idx].reshape(8, 8), cmap='gray')
    axes[2].set_title(f'预测：{y_pred[idx]}, 真实：{y_true[idx]}')
    axes[2].axis('off')

plt.tight_layout()
plt.savefig('../images/神经网络 - 数字识别.png', dpi=150, bbox_inches='tight')
plt.show()
```
