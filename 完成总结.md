## 🚀 学习建议

### 初学者路线
```
第 0 章 → 第 1 章 (线性回归→逻辑回归) → 第 4 章 (神经网络)
   ↓
第 2 章 (聚类/降维) → 第 3 章 (评估) → 实战项目
```

### 有基础路线
```
直接学习感兴趣的算法章节 → 实战项目 → 深入学习
```

### 复习路线
```
看每章的"核心要点"小结 → 运行代码示例 → 做实战项目
```
