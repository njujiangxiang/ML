# 2. 层次聚类：树状分组

## 🎯 学习目标
- 理解层次聚类的核心思想
- 掌握凝聚和分裂两种策略
- 学会解读树状图 (Dendrogram)
- 理解不同距离度量方法的影响

---

## 一、层次聚类 vs K-Means

### 核心区别

```
K-Means:
- 需要指定 K 值
- 直接分成 K 组
- 扁平结构

  ●●●  ○○○
  ●●●  ○○○
  (K=2)

层次聚类:
- 不需要指定 K
- 生成树状结构
- 可以看不同粒度

      ┌───┴───┐
    ┌─┴─┐   ┌─┴─┐
    ●●●   ○○○
  (可以切在任意高度)
```

### 层次聚类的优势

```
✓ 不需要预先指定 K
✓ 生成有意义的树状结构
✓ 可以发现任意形状的聚类
✓ 可以可视化聚类过程
✓ 适合小数据集

✗ 计算复杂度高 O(n³)
✗ 不适合大数据集
✗ 一旦合并/分裂，不可撤销
```

---

## 二、两种策略

### 1. 凝聚聚类 (Agglomerative)

```
自底向上：每个点是一类，逐步合并

步骤：
1. 每个样本作为一个聚类
2. 合并最近的两个聚类
3. 重复 2，直到所有样本合并为一类

可视化：
  ●  ○  ●  ○  ●    初始：5 个聚类
  │   │   │   │   │
  └───┘   │   └───┘   合并最近的
    │     │     │
    └─────┴─────┘   继续合并
          │
        最终：1 个聚类
```

### 2. 分裂聚类 (Divisive)

```
自顶向上：所有点是一类，逐步分裂

步骤：
1. 所有样本作为一个聚类
2. 分裂成两个最不同的聚类
3. 重复 2，直到每个样本一类

可视化：
  ●●●●●    初始：1 个聚类
    │
  ┌─┴─┐
●●●  ●●    分裂成 2 个
 │    │
┌┴┐  ┌┴┐   继续分裂
● ●  ● ●   最终：5 个聚类
```

### 常用策略

```
凝聚聚类更常用！
- 实现简单
- 计算效率高 (相对分裂)
- scikit-learn 只实现了凝聚

分裂聚类：
- 理论上更好
- 计算复杂
- 很少使用
```

## 三、距离度量

### 样本间距离

```
1. 欧氏距离 (默认)
   d(x,y) = √Σ(xᵢ-yᵢ)²

2. 曼哈顿距离
   d(x,y) = Σ|xᵢ-yᵢ|

3. 余弦距离
   d(x,y) = 1 - (x·y)/(||x||·||y||)
   
   适合：文本、高维稀疏数据
```

### 聚类间距离 (Linkage)

```
关键：如何定义两个聚类的距离？

1. 单连接 (Single Linkage)
   最近邻距离
   
   d(C₁,C₂) = min(d(x,y))
              x∈C₁, y∈C₂
   
   ●─●       ○─○
   │←─────→│    最近的两点
   
   特点：
   ✓ 可以发现链状结构
   ✗ 链式效应 (chaining)

2. 全连接 (Complete Linkage)
   最远邻距离
   
   d(C₁,C₂) = max(d(x,y))
              x∈C₁, y∈C₂
   
   ●─●       ○─○
   │←─────────→│  最远的两点
   
   特点：
   ✓ 紧凑的球形聚类
   ✗ 对异常值敏感

3. 平均连接 (Average Linkage)
   所有点对的平均距离
   
   d(C₁,C₂) = (1/n₁n₂) ΣΣ d(x,y)
                     x∈C₁ y∈C₂
   
   ●─●       ○─○
   ←───平均──→
   
   特点：
   ✓ 平衡单连接和全连接
   ✓ 最常用

4. 质心连接 (Centroid Linkage)
   聚类质心的距离
   
   d(C₁,C₂) = d(centroid₁, centroid₂)
   
       ●         ○
       ★─────★   质心
   
   特点：
   ✓ 计算快
   ✗ 可能逆序 (合并后距离反而小)

5. Ward 法 (默认推荐)
   最小化方差增加
   
   合并后总方差增加最小的两个聚类
   
   特点：
   ✓ 紧凑的球形聚类
   ✓ 最常用
   ✗ 只能配合欧氏距离
```

## 四、完整代码实现

### 使用 scikit-learn

```python
from sklearn.cluster import AgglomerativeClustering
from sklearn.datasets import make_blobs
import matplotlib.pyplot as plt
import numpy as np

# 生成数据
X, y = make_blobs(n_samples=200, centers=4, 
                  cluster_std=0.60, random_state=42)

# 创建模型
model = AgglomerativeClustering(
    n_clusters=4,           # 聚类数量 (None 表示不指定)
    affinity='euclidean',   # 距离度量
    linkage='ward'          # 连接方法：ward/complete/average/single
)

# 训练 (拟合 + 预测)
labels = model.fit_predict(X)

print(f"聚类数量：{len(np.unique(labels))}")
print(f"每个聚类的样本数：{np.bincount(labels)}")

# 可视化
plt.figure(figsize=(10, 5))

plt.subplot(1, 2, 1)
plt.scatter(X[:, 0], X[:, 1], c=y, cmap='viridis', s=50)
plt.title('真实标签')

plt.subplot(1, 2, 2)
plt.scatter(X[:, 0], X[:, 1], c=labels, cmap='viridis', s=50)
plt.title(f'层次聚类 (linkage={model.linkage})')

plt.tight_layout()
plt.savefig('../images/层次聚类 - 示例.png', dpi=150, bbox_inches='tight')
plt.show()
```

### 绘制树状图

```python
from scipy.cluster.hierarchy import dendrogram, linkage
import matplotlib.pyplot as plt

# 计算连接矩阵
Z = linkage(X, method='ward')  # method: ward/complete/average/single

# 绘制树状图
plt.figure(figsize=(12, 6))
dendrogram(Z, leaf_rotation=90, leaf_font_size=8)
plt.xlabel('样本索引')
plt.ylabel('距离')
plt.title('层次聚类树状图')
plt.axhline(y=10, color='r', linestyle='--', label='切割线 (K=4)')
plt.legend()
plt.savefig('../images/层次聚类 - 树状图.png', dpi=150, bbox_inches='tight')
plt.show()

# 从树状图读取 K 值
# 找最长的垂直线，画水平线穿过
# 穿过的垂直线数量 = K
```

### 不同连接方法对比

```python
from sklearn.cluster import AgglomerativeClustering
import matplotlib.pyplot as plt

methods = ['ward', 'complete', 'average', 'single']

fig, axes = plt.subplots(2, 2, figsize=(12, 10))

for ax, method in zip(axes.flat, methods):
    model = AgglomerativeClustering(
        n_clusters=4,
        affinity='euclidean' if method != 'ward' else None,
        linkage=method
    )
    labels = model.fit_predict(X)
    
    ax.scatter(X[:, 0], X[:, 1], c=labels, cmap='viridis', s=50)
    ax.set_title(f'Linkage: {method}')

plt.tight_layout()
plt.savefig('../images/层次聚类 - 连接方法对比.png', dpi=150, bbox_inches='tight')
plt.show()
```

## 五、选择聚类数量

### 树状图法

```
思想：找最长的垂直距离

树状图：
距离
 ↑
 │          │
 │          │  ← 长垂直线
 │      ┌───┴───┐
 │      │       │
 │  ┌───┴───┐   │
 │  │       │   │
 │──┴─┴──┴─┴───┴──
 └──────────────→ 样本

在最长的垂直线处切割
穿过的线数 = K
```

### 轮廓系数法

```python
from sklearn.metrics import silhouette_score
import numpy as np

# 尝试不同的 K
k_range = range(2, 11)
silhouette_scores = []

for k in k_range:
    model = AgglomerativeClustering(n_clusters=k, linkage='ward')
    labels = model.fit_predict(X)
    score = silhouette_score(X, labels)
    silhouette_scores.append(score)
    print(f"K={k}, 轮廓系数={score:.4f}")

# 绘制
plt.figure(figsize=(10, 6))
plt.plot(k_range, silhouette_scores, 'o-')
plt.xlabel('聚类数量 K')
plt.ylabel('轮廓系数')
plt.title('轮廓系数 vs K')
plt.grid(True, alpha=0.3)
plt.savefig('../images/层次聚类 -K 选择.png', dpi=150, bbox_inches='tight')
plt.show()

# 最佳 K
best_k = k_range[np.argmax(silhouette_scores)]
print(f"\n最佳 K 值：{best_k}")
```

### 距离跳跃法

```python
from scipy.cluster.hierarchy import linkage

# 计算连接矩阵
Z = linkage(X, method='ward')

# 获取每次合并的距离
distances = Z[:, 2]

# 计算距离的跳跃 (二阶差分)
jumps = np.diff(np.diff(distances))

# 最大跳跃的位置
max_jump_idx = np.argmax(jumps) + 2  # +2 因为二阶差分

# 对应的 K 值
n_samples = len(X)
best_k = n_samples - max_jump_idx

print(f"基于距离跳跃法的最佳 K: {best_k}")

# 可视化
plt.figure(figsize=(10, 6))
plt.plot(range(2, len(distances)+1), distances, 'o-')
plt.xlabel('合并步骤')
plt.ylabel('合并距离')
plt.title('合并距离图')
plt.axvline(x=max_jump_idx, color='r', linestyle='--', 
           label=f'最大跳跃点 (K≈{best_k})')
plt.legend()
plt.grid(True, alpha=0.3)
plt.savefig('../images/层次聚类 - 距离跳跃.png', dpi=150, bbox_inches='tight')
plt.show()
```

## 六、实战案例：基因表达数据分析

```python
import pandas as pd
import numpy as np
from sklearn.cluster import AgglomerativeClustering
from sklearn.preprocessing import StandardScaler
from scipy.cluster.hierarchy import dendrogram, linkage
import matplotlib.pyplot as plt
import seaborn as sns

# 1. 加载数据 (模拟基因表达数据)
np.random.seed(42)
n_genes = 100
n_samples = 50

# 模拟 4 种不同类型的基因表达模式
patterns = [
    np.random.normal(10, 1, n_samples),      # 高表达
    np.random.normal(5, 1, n_samples),       # 中表达
    np.random.normal(0, 1, n_samples),       # 低表达
    np.random.normal(5, 2, n_samples)        # 变异大
]

# 组合成表达矩阵
X = np.vstack([
    np.random.choice(patterns, size=25) for _ in range(n_genes)
]).reshape(n_genes, n_samples)

print(f"数据形状：{X.shape}")
print(f"基因数：{n_genes}")
print(f"样本数：{n_samples}")

# 2. 数据标准化
scaler = StandardScaler()
X_scaled = scaler.fit_transform(X.T).T  # 对样本标准化

# 3. 层次聚类 (对基因)
Z_genes = linkage(X_scaled, method='ward')

# 4. 层次聚类 (对样本)
Z_samples = linkage(X_scaled.T, method='ward')

# 5. 绘制热图 + 树状图
plt.figure(figsize=(14, 10))

# 创建网格
from scipy.cluster.hierarchy import leaves_list

gene_order = leaves_list(Z_genes)
sample_order = leaves_list(Z_samples)

# 排序后的数据
X_sorted = X_scaled[gene_order][:, sample_order]

# 绘制热图
sns.heatmap(X_sorted, cmap='RdYlBu_r', center=0,
            cbar_kws={'label': '标准化表达量'})
plt.xlabel('样本')
plt.ylabel('基因 (已排序)')
plt.title('基因表达热图 (层次聚类排序)')
plt.tight_layout()
plt.savefig('../images/层次聚类 - 基因热图.png', dpi=150, bbox_inches='tight')
plt.show()

# 6. 绘制双树状图
fig = plt.figure(figsize=(14, 10))

# 基因树状图 (上方)
ax1 = fig.add_axes([0.1, 0.6, 0.8, 0.3])
dendrogram(Z_genes, orientation='top', ax=ax1,
           leaf_rotation=90, leaf_font_size=8)
ax1.set_title('基因聚类')
ax1.set_ylabel('距离')

# 样本树状图 (右侧)
ax2 = fig.add_axes([0.9, 0.1, 0.05, 0.5])
dendrogram(Z_samples, orientation='right', ax=ax2,
           leaf_rotation=0, leaf_font_size=8)
ax2.set_title('样本聚类')
ax2.set_xlabel('距离')

# 热图 (中间)
ax3 = fig.add_axes([0.1, 0.1, 0.8, 0.5])
im = ax3.imshow(X_sorted, aspect='auto', cmap='RdYlBu_r')
ax3.set_xlabel('样本')
ax3.set_ylabel('基因')
plt.colorbar(im, ax=ax3, label='标准化表达量')

plt.savefig('../images/层次聚类 - 双树状图.png', dpi=150, bbox_inches='tight')
plt.show()

# 7. 切割树状图得到聚类
from scipy.cluster.hierarchy import fcluster

# 切割基因树状图，得到 4 个聚类
gene_clusters = fcluster(Z_genes, t=4, criterion='maxclust')
print(f"\n基因聚类结果:")
for i in range(1, 5):
    n_genes_in_cluster = np.sum(gene_clusters == i)
    print(f"  聚类{i}: {n_genes_in_cluster} 个基因")

# 分析每个聚类的表达模式
fig, axes = plt.subplots(2, 2, figsize=(12, 8))
for ax, i in zip(axes.flat, range(1, 5)):
    cluster_genes = X_scaled[gene_clusters == i]
    mean_expression = np.mean(cluster_genes, axis=0)
    std_expression = np.std(cluster_genes, axis=0)
    
    ax.plot(mean_expression, label='均值')
    ax.fill_between(range(n_samples), 
                    mean_expression - std_expression,
                    mean_expression + std_expression,
                    alpha=0.3, label='±标准差')
    ax.set_title(f'聚类{i} (n={np.sum(gene_clusters==i)})')
    ax.set_xlabel('样本')
    ax.set_ylabel('表达量')
    ax.legend()
    ax.grid(True, alpha=0.3)

plt.tight_layout()
plt.savefig('../images/层次聚类 - 聚类模式.png', dpi=150, bbox_inches='tight')
plt.show()
```

## 七、优缺点总结

### 优点 ✅

```
✓ 不需要指定 K 值
  - 可以从树状图选择
  - 灵活

✓ 生成树状结构
  - 有生物学意义 (如进化树)
  - 可视化好

✓ 可以发现任意形状
  - 不假设球形
  - 适合复杂数据

✓ 适用于小数据集
  - <1000 样本效果好
```

### 缺点 ❌

```
❌ 计算复杂度高
  - O(n³) 时间
  - O(n²) 空间
  - 不适合大数据

❌ 不可撤销
  - 一旦合并/分裂，无法回退
  - 局部最优

❌ 对噪声敏感
  - 异常值影响大
  - 尤其是单连接

❌ 需要选择连接方法
  - 不同方法结果不同
  - 需要尝试
```
