## 💡 核心概念卡片

### 梯度下降
```
直觉：下山找最低点
公式：w = w - lr × gradient
关键：学习率要合适
```

### 过拟合 vs 欠拟合
```
欠拟合：模型太简单 → 增加复杂度
过拟合：模型太复杂 → 正则化/剪枝/更多数据
```

### 交叉熵损失
```
用于分类问题
预测越错，惩罚越大
比 MSE 更适合分类
```

## ❓ 常见问题

### Q: 为什么需要标准化？
A: 不同特征量纲不同，标准化后梯度下降更快收敛

### Q: 如何选择学习率？
A: 试几个值 (0.001, 0.01, 0.1)，看哪个损失下降最快

### Q: 过拟合了怎么办？
A: 
1. 增加数据
2. 减少特征
3. 增加正则化
4. 早停

## 🔖 重要公式

### 线性回归
```
模型：y = w·x + b
损失：MSE = (1/n) × Σ(y - ŷ)²
梯度：∂L/∂w = (-2/n) × Σ(y - ŷ) × x
```

### 逻辑回归
```
模型：P = 1 / (1 + e^(-w·x-b))
损失：CE = -[y·log(P) + (1-y)·log(1-P)]
```

### 决策树
```
熵：H = -Σ p·log₂(p)
信息增益：Gain = H(父) - H(子)
基尼：Gini = 1 - Σ p²
```
